This project is a system developed in Python that allows you to easily manage student information from the console.

It is designed to register, query, update, and delete students, facilitating the organization of academic data.

Requirements to run it:
Python 3.8 or higher installed
A code editor such as VS Code
Navigate to the project folder and run it.

The program has the following options:
Register student: This allows you to add a new student by entering data such as: name, age, ID, course.
View students: Displays the list of all registered students.
Search student: Allows you to search for a student by their ID or name.
Update student: Modifies the information of an existing student.
Delete student: Removes a student from the system using their ID.

Data can be stored in a .json file or in memory structures (lists/dictionaries).

Example of use: Register student
Enter name: Juan Perez
Enter age: 20
Enter ID: 12345
Enter course: Mathematics
"Student successfully registered"

Example of use: View student
List of students:
ID: 12345 | Name: Juan Perez | Age: 20 | Course: Mathematics
